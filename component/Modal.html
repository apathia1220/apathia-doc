Modal 模态窗 #

在保留当前页面状态的情况下，告知用户并承载相关操作。

基础用法 #

弹出一个对话框

template用法（不推荐） #

TIP

modal 模态窗组件推荐使用 function 方式.有益于逻辑抽离和后续维护。

需要设置 model-value / v-model 属性，它接收 Boolean，当为 true 时显示 Modal。

<template>
  <div>
    <Modal v-model="show" title="template基础用法">
      我们的孤独就像天空中漂浮的城市，仿佛是一个秘密，却无从述说。
    </Modal>
    <BaseButton @click="showModal">基本用法</BaseButton>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const show = ref(false)
const showModal = () => {
  show.value = true
}
</script>

function 用法（推荐） #

TIP

使用 function 方式需要全局注册使用 modalInstall 方法

import { modalInstall } from '@apathia/apathia'
app.use(modalInstall)

<template>
  <div>
    <BaseButton @click="showModalFn">function 基础用法</BaseButton>
  </div>
</template>

<script setup>
import { getCurrentInstance, h } from 'vue'

const { proxy } = getCurrentInstance()
const { modal, remove } = proxy.useModal()

const showModalFn = () => {
  modal({
    title: '函数式调用',
    render: () =>
      h(
        'div',
        '一个人的时候，怎么过？就算再难以承受，其实到最后我们总能走得过来。',
      ),
  })
}
</script>

自定义 header #

可以完全自定义header的头部

template 用法 #

template 方式通过 header 插槽实现，同时返回关闭modal 的函数 close

<template>
  <div>
    <Modal v-model="show" title="template基础用法">
     <template #header="{ close }">
       <div :class="contentClass">
         <span>自定义header</span>
         <BaseButton danger @click="close">关闭</BaseButton>
       </div>
     </template>
      <template #default >
        <div>我们的孤独就像天空中漂浮的城市，仿佛是一个秘密，却无从述说。</div>
      </template>
    </Modal>
    <BaseButton @click="showModal">自定义 header </BaseButton>
  </div>
</template>

<script setup>
import { getCurrentInstance, ref } from 'vue'
const {proxy} = getCurrentInstance()
const { style } = proxy
const contentClass = style`flex w-full justify-between`
const show = ref(false)
const showModal = () => {
  show.value = true
}
</script>

function 用法 #

function 方式通过 renderHeader 方法实现, 返回值为 h 函数 、字符 、JSX等

<template>
  <div>
    <BaseButton @click="showModalFn">自定义header</BaseButton>
  </div>
</template>

<script setup>
import { getCurrentInstance, h } from 'vue'

const { proxy, BaseButton } = getCurrentInstance()
const { modal, remove } = proxy.useModal()

const showModalFn = () => {
  modal({
    render: () =>
      h(
        'div',
        '一个人的时候，怎么过？就算再难以承受，其实到最后我们总能走得过来。',
      ),
    renderHeader: () => 
      h(
        'div',
        [
          h(
            'sapn',
            'function 自定义header'
          )
        ]
      )
  })
}
</script>

嵌套使用 #

<template>
  <BaseButton @click="showModalFn1">嵌套使用</BaseButton>
</template>

<script>
import { defineComponent, getCurrentInstance, h } from 'vue'

export default defineComponent({
  setup() {
    const { proxy } = getCurrentInstance()
    const { modal, remove } = proxy.useModal()

    const showModalFn = () => {
      modal({
        title: '函数式调用 - 内层',
        render: () =>
          h(
            'div',
            '一个人的时候，怎么过？就算再难以承受，其实到最后我们总能走得过来。',
          ),
      })
    }

    const showModalFn1 = () => {
      modal({
        title: '函数式调用 - 外层',
        width: 700,
        render: () => h('div', { onClick: showModalFn }, '点击打开二级modal'),
      })
    }
    return {
      showModalFn1,
    }
  },
})
</script>

Modal props #

prop说明类型默认值
modelValue是否展示, template 方式使用booleanfalse
title标题string''
subTitle副标题string''
top距离页面顶部距离number,string60(px)
width模态窗宽度number,string640(px)
showClose是否展示右上角关闭按钮booleantrue
maskClosable是否可以点击蒙层关闭booleanfalse
keyboard是否可以通过 Esc 按键关闭booleantrue
beforeClose关闭前校验function: () => bool.bool 为true则关闭，否则阻止关闭() => true
renderfunction 方式的内容render函数function: () => stringh
renderHeaderfunction 方式的header的render函数function: () => stringh

名称说明接收参数
-default(默认插槽)，自定义内容插槽-
header自定义头部close: 关闭modal

events #

close: 关闭时触发。无参数
